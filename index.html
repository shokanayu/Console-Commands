CLASS METHOD VARIABLES OPTIONS DESCRIPTION








InputRebindSetting ClearBindings


Sets all your keybinds found in settings to an unset state

InteractionHandler PlayInteraction interaction identifier
AbilityUnlock_Captain Runs a conversation with an NPC

InteractionHandler PrintRetrievableInteractions


Prints to the console all interactions that have their requirements currently met (excludes non-repeatable, already seen interactions)

InteractionHandler ShowNextInteraction


Plays the next interaction in line

Items AddItem item identifier
Active_Boost Adds an item to your inventory

Items OpenItemsUI


Makes Gan's items unlocked screen visible

Items RandomItemStatistics


Prints to the console the current chances for item rarities to appear

Items ShowAllItems


Sets the _ShowItem fact to true for all items. These facts are only used for the achievement, "Discover All Items"

Items UnlockAll


Sets the item_unlocked_ fact to true for all items

Items UnlockAndShowAll


Runs the commands, "UnlockAll" and "ShowAllItems"

AchievementHandler Clear


Resets progress for all achievements, and removes any you had already achieved

AchievementHandler Unlock

INVALID Grants you the specified achievement

GameLoop AddLife


Returns a missing heart to Zoe (Only works during a run)

GameLoop AddSparks spark amount

Gives the specified amount of sparks

GameLoop Heal health amount

Refills Zoe's health bar by the specified amount

GameLoop RemoveLife


Takes a heart from Zoe (Only works during a run)

GameLoop SkipLevel


Skips the current level, if applicable

GameLoop TakeHalfHealthDmg


Reduces Zoe's health bar by half it's current value (Currently breaks the health bar visually, making it appear completely black)

GameLoop TestModal


Makes a test UI prompt visible

FactSystem AddToFact fact name fact value tutorial_finished Adds the specified value to the specified fact

FactSystem DeleteFact fact name
Removes the specified fact from the fact list

FactSystem GetFact fact name
Returns the value of the specified fact as a standalone value

FactSystem PrintFact fact name
Returns the value of the specified fact in a human readable way

FactSystem SetFact fact name fact value Adds the specified fact to the fact list with the specified value, or alters the specified fact if it already exists in the fact list

MetaProgression ActivateAbility


Selects the specified ability to become your active ability

MetaProgression AddResource antispark amount

Gives the specified amount of antisparks

MetaProgression OpenUI


Makes the Captain's upgrade menu visible

MetaProgression Unlock ability identifier
BoardBoost Grants access to use the specified ability

LevelSelectionHandler PlayNode node number

Plays the specified node in the current run

PlayerData SetCurrentEnergy energy amount

Overrides Zoe's current energy value with the specified amount

PlayerData SetCurrentHealth health amount

Overrides Zoe's current health value with the specified amount

PlayerData SetEnergyBars bar amount

Splits Zoe's energy bar into the specified amount of segments (Currently only works when used in the map screen, as the game overrides it while running)

PlayerData SetSparks spark amount

Override's your current spark count with the specified amount

Player AddEnergyCmd energy amount

Fills up Zoe's energy bar by the specified amount

PlayerProgress Reset


Sets your shards completed to 0

PlayerProgress SetShardComplete shard id
0 Marks the specified shard as beaten

PlayerProgress UnlockAll


Marks all shards as beaten

PlayerProgress UnlockToShard shard id
0 Marks all shards before the specified shard as beaten

RunHandler LoseRun transition override boolean

Abandons the current run shard, if applicable

RunHandler PlayEncounter encounter identifier
sage_01_planting Transitions to the specified encounter (Doesn't work correctly if used in the hub)

RunHandler PlayRestScene


Transitions to the campfire fragment (Portals don't work when used from the hub)

RunHandler WinRun transition override boolean

Completes the current run shard, if applicable

SaveSystem Clear


Zeroes your current save file and erases loaded data

SaveSystem Load


Replaces your currently loaded data with your save file's data

SaveSystem LoadTempSave


Replaces your current data with an empty save file template

SaveSystem OverwriteSave json string with save data

Replaces your current save file with the specified data

SaveSystem Save


Updates your current save file with your current data

SaveSystem SaveToTempSave


Updates your temporary save file with your current data

TutorialPopupHandler DebugOpenPopup tutorial identifier
FindShard Makes the specified tutorial visible

WorldShard PlayShard shard id
0 Immediately enters a run of the specified shard with a random seed

WorldShard PlayShardSeeded shard id shard seed 0 Immediately enters a run of the specified shard with the specified seed

Application SetTargetFramerate frame rate

Makes the game attempt to run at the specified frame rate

Console Clear


Deletes all the history of the console

Console ClearMuted


Deletes only the muted message history of the console

Console Pause


Stops the console's message logging

Console SetDPI dpi value

Changes the game's cursor DPI to the specified value

Console Unpause


Reumes the console's message logging, if it is currently paused

Script Execute file path to script

Runs a script at the specified path
